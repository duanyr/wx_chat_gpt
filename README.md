# wx_gpt_chat

## 具体功能

- 微信公众号ChatGPT聊天窗口实现 
- 应用openAI的流式数据获取能力，结合客服消息实现消息对话流畅体验


## 使用方法

1. 因为该功能是与openAI对接，最好准备一台欧洲或美洲的服务器，阿里云和腾讯云均可购买
2. 服务器配置好centos或ubuntu系统环境，将代码clone到主机上
4. 运行服务
